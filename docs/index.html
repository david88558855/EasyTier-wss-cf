EasyTier WebSocket Relay

A serverless, secure and high-performance WebSocket (WSS) relay for EasyTier mesh VPN, deployed globally on Cloudflare Workers & Durable Objects.

Deploy to Cloudflare Setup Guide

Serverless & Free

Runs entirely on Cloudflare free tier infrastructure. No VPS, domain names, or physical servers required.

Zero Configuration

No external databases needed. Autopilot registration, configurations and client tokens are safely stored inside Durable Objects.

Visual Admin

A premium built-in management console to monitor node connections, manage client tokens, and draw real-time topology.

5 Languages

Fully localized for international usage. Switch seamlessly between English, Simplified Chinese, Traditional Chinese, Japanese, and Korean.

How to Use

1

Fork & Deploy

Fork this repository, connect it in your Cloudflare Workers dashboard, and deploy. The setup is 100% automated.

2

Access Admin Dashboard

Navigate to your deployment URL in the browser (e.g., https://your-worker.workers.dev) and sign in using the default password admin.

3

Connect Client

Specify the WSS address as a peer in your EasyTier configuration. Use the default room or create a custom one.

wss://your-deployment.workers.dev:0/ws?room=my_network
